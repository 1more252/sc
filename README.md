# SCARLETT — Voice AI Assistant (AI Upgrade)

## Что нового в этой версии

### Апгрейд 1: Долгосрочная векторная память (ChromaDB)
Scarlett теперь помнит тебя между сессиями.

**Как работает:**
- Все разговорные фразы пользователя сохраняются в локальную ChromaDB
- При каждом AI-запросе ищутся топ-3 семантически похожих воспоминания
- Находки добавляются в системный промпт: "Из долгосрочной памяти: ..."
- AI отвечает с учётом того что ты говорил раньше
- Хранится в `~/.scarlett_memory/`, работает полностью оффлайн

**Пример:**
- Сегодня ты сказал: "я работаю программистом на Python"
- Через неделю спрашиваешь "посоветуй IDE"
- Scarlett вспоминает что ты питонист и советует PyCharm/VS Code

**Требования:** `chromadb` + `sentence-transformers` (устанавливаются через install.bat)

---

### Апгрейд 2: Параллельное выполнение команд
"Открой телеграм и включи музыку" — обе команды запускаются одновременно.

**Как работает:**
- Парсер ищет разделители: "и", "а также", "плюс"
- Если найдены — запускает каждую часть в отдельном потоке
- Результаты объединяются в один ответ
- Работает без дополнительных зависимостей

**Примеры фраз:**
- "открой хром и включи youtube"
- "тише и открой дискорд"
- "пауза и найди котиков"

---

### Апгрейд 3: Системный контекст в реальном времени
AI знает что происходит на твоём компьютере прямо сейчас.

**Что видит Scarlett:**
- Активное окно (win32gui) — "активное окно: PyCharm"
- Запущенные приложения (psutil) — "открыто: Steam, Discord"
- Текущая музыка (Windows Media API) — "играет: трек — артист"
- Время суток — "вечер"

**Пример:**
- Steam открыт → AI: "Вижу ты в Steam. Во что играешь?"
- VS Code активен → AI знает что ты кодишь и отвечает соответственно

**Кэш:** данные обновляются раз в 5 секунд — не нагружает систему

---

### Апгрейд 4: LLM Tool Calling
Вместо 500+ строк if/elif — LLM сам понимает команду и вызывает нужный инструмент.

**Доступные инструменты (tools):**
- `open_app` — открыть приложение, игру, сайт
- `set_volume` — установить громкость
- `search_web` — найти в интернете
- `set_timer` — поставить таймер/напоминание
- `media_control` — управление музыкой

**Как включить:** нужен API ключ Groq или Mistral в Настройках (⚙)
- Groq: бесплатно, console.groq.com (нужен VPN в РФ)
- Mistral: бесплатно, console.mistral.ai (работает в РФ без VPN)

**Graceful degradation:** если ключа нет или LLM недоступен — автоматически
работает стандартная обработка команд (if/elif), без изменений для пользователя.

## Настройки AI-апгрейдов
В файле `%USERPROFILE%\.scarlett_config.json`:

```json
{
  "vector_memory_enabled": true,
  "context_enabled": true,
  "tool_calling_enabled": true,
  "groq_key": "gsk_...",
  "mistral_key": "..."
}
```

Или через кнопку ⚙ в интерфейсе.
